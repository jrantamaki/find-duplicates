# find-duplicates

A Clojure hack to find dulicate files in a fiven directory

## Usage

Package with Leiningen

	lein uberjar

And then run with:

        java -jar target/find-duplicates-0.1.0-SNAPSHOT-standalone.jar $HOME/myfiles

Or with Leiningen:

	lein run  /home/jarkko/myfiles
